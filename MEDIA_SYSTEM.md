# Система Управления Медиа - CMS Pro

## Описание
Система управления медиа файлами в админ-панели с поддержкой загрузки фотографий и видео.

## Поддерживаемые Форматы

### ✅ Изображения
- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)

### ✅ Видео
- **MP4** (.mp4)
- **WebM** (.webm)
- **OGG** (.ogv)
- **QuickTime** (.mov)

### ❌ Не поддерживается
- PDF, Word, Excel и другие документы
- Архивы и другие форматы
- Аудиофайлы

## Как использовать

### 1. Загрузка файлов

1. Перейти в **Админ-панель** → **Медиа**
2. Нажать кнопку **"Загрузить файлы"**
3. Выбрать один или несколько файлов
4. Файлы будут загружены и добавлены в библиотеку

### 2. Просмотр загруженных файлов

- **Изображения**: показываются как миниатюры
- **Видео**: показываются с видео-плеером для предпросмотра
- Каждый файл отображает:
  - Название файла
  - Размер файла
  - Тип (Изображение/Видео)
  - Дату загрузки

### 3. Управление файлами

Для каждого файла доступны действия:
- **Скачать** - загрузить файл на компьютер
- **Удалить** - удалить файл из библиотеки

## Валидация Файлов

При попытке загрузить неподдерживаемый формат:
- Система покажет диалоговое окно с ошибкой
- Будут перечислены неподдерживаемые файлы
- Поддерживаемые файлы все равно будут загружены

**Пример ошибки:**
```
⚠️ Ошибка!

Поддерживаемые типы файлов:
• Изображения: JPEG, PNG
• Видео: MP4, WebM, OGG, MOV

Неподдерживаемые файлы:
presentation.pdf
document.docx
```

## Технические Детали

### Хранение
- Файлы хранятся в `localStorage` с ключом `mediaFiles`
- Превью (thumbnail) сохраняется в виде Data URL
- Максимальный размер ограничен возможностями localStorage (зависит от браузера, обычно 5-10 MB)

### Структура данных медиа-файла
```javascript
{
  id: 1716000000000,              // Уникальный ID (timestamp)
  name: "photo.jpg",              // Имя файла
  type: "image/jpeg",             // MIME тип
  size: 245760,                   // Размер в байтах
  preview: "data:image/jpeg;...", // Data URL превью
  uploadDate: "2026-06-17T...",   // Дата загрузки (ISO)
  isImage: true,                  // Флаг типа
  isVideo: false                  // Флаг типа
}
```

### JavaScript API

#### `initMediaStorage()`
Инициализирует хранилище медиа файлов

#### `getAllMediaFiles()`
Возвращает массив всех загруженных файлов

#### `saveMediaFile(file, preview)`
Сохраняет новый медиа-файл
- **Параметры:**
  - `file` - File объект
  - `preview` - Data URL превью
- **Возвращает:** объект медиа-файла или null

#### `isValidMediaFile(file)`
Проверяет, является ли файл поддерживаемого типа
- **Параметры:** `file` - File объект
- **Возвращает:** boolean

#### `getFileTypeLabel(mimeType)`
Возвращает человекочитаемое название типа файла
- **Параметры:** `mimeType` - MIME тип
- **Возвращает:** строка ("Изображение", "Видео", "Неизвестно")

#### `formatFileSize(bytes)`
Форматирует размер файла в читаемый формат
- **Параметры:** `bytes` - размер в байтах
- **Возвращает:** строка ("2.4 MB", "1.8 KB", и т.д.)

#### `deleteMediaFile(fileId)`
Удаляет медиа-файл из хранилища
- **Параметры:** `fileId` - ID файла

## Ограничения

### Размер файлов
- Максимальный размер зависит от браузера и доступного хранилища
- Рекомендуется: до 10 MB на файл
- Общее хранилище: обычно 5-10 MB на домен

### Количество файлов
- Не ограничено, но зависит от размера хранилища

### Браузеры
- Chrome/Edge: 10 MB
- Firefox: 10 MB
- Safari: 5 MB
- Opera: 10 MB

## Для Production

Текущая система использует `localStorage`, что подходит только для разработки.

Для production рекомендуется:
1. **Сервер** - загружайте файлы на сервер
2. **Облачное хранилище** - используйте AWS S3, Google Cloud Storage и т.д.
3. **CDN** - распределите контент через CDN для быстрого доступа
4. **Обработка изображений** - сжимайте и оптимизируйте изображения
5. **Ограничения** - установите ограничения на размер и типы файлов на сервере

## Примеры использования

### Получить все загруженные файлы
```javascript
const files = getAllMediaFiles();
console.log(files);
```

### Получить только изображения
```javascript
const images = getAllMediaFiles().filter(f => f.isImage);
```

### Получить только видео
```javascript
const videos = getAllMediaFiles().filter(f => f.isVideo);
```

### Получить файл по ID
```javascript
const files = getAllMediaFiles();
const file = files.find(f => f.id === 1716000000000);
```

### Удалить все файлы
```javascript
const files = getAllMediaFiles();
files.forEach(f => deleteMediaFile(f.id));
```

## Troubleshooting

### Файл не загружается
- Проверьте, что формат файла поддерживается
- Проверьте размер файла (не более 10 MB на файл)
- Проверьте свободное место в localStorage браузера

### Предпросмотр не показывается
- Для видео: проверьте, что браузер поддерживает видео-плеер
- Для изображений: проверьте, что файл не повреждена

### Файлы исчезли после перезагрузки
- Проверьте, что localStorage не отключена в браузере
- Проверьте, что сайт работает с протоколом HTTPS (в некоторых браузерах требуется)

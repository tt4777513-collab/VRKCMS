## 📄 Управление страницами

### Создать новую страницу
1. Admin Panel → **Страницы** → **Новая страница**
2. Заполните форму:
   - Название: `Мой заголовок`
   - URL Slug: `moy-zagolovok` (автоматически)
   - Содержание: введите текст
3. Нажмите **Сохранить**

### Отредактировать страницу
1. Admin Panel → **Страницы**
2. Найдите страницу в таблице
3. Нажмите кнопку **Edit** (карандаш)
4. Измените нужные поля
5. Нажмите **Сохранить**

### Опубликовать страницу
1. Отредактируйте страницу
2. В правой колонке установите **Статус: Опубликована**
3. Нажмите **Сохранить**

### Удалить страницу
1. Admin Panel → **Страницы**
2. Нажмите кнопку **Delete** (корзина)
3. Подтвердите удаление

## ✏️ Редактирование контента

### Форматирование текста

| Что | Как | Горячая клавиша |
|-----|-----|-----------------|
| Жирный | Выделите текст и нажмите **B** | Ctrl+B |
| Курсив | Выделите текст и нажмите *I* | Ctrl+I |
| Заголовок | Выделите текст и нажмите **H** | Ctrl+H |
| Список | Нажмите **•** | Ctrl+L |
| Ссылка | Выделите и нажмите **🔗**, вставьте URL | Ctrl+K |

### Вставить изображение
1. Нажмите кнопку **🖼️** в редакторе
2. Выберите изображение из библиотеки или загрузите новое
3. Введите alt-текст (для SEO)
4. Нажмите **OK**

### Вставить видео
1. Нажмите кнопку **🎬** (если есть)
2. Введите YouTube URL: `https://www.youtube.com/watch?v=...`
3. Нажмите **OK**

## 📸 Медиа библиотека

### Загрузить файлы
1. Admin Panel → **Медиа** → **Загрузить файлы**
2. Выберите файлы на компьютере (можно несколько)
3. Нажмите **Загрузить**

### Поддерживаемые форматы
- **Изображения**: JPG, PNG, GIF, WebP, SVG
- **Видео**: MP4, WebM, OGG
- **Документы**: PDF, DOC, DOCX, XLS, XLSX
- **Архивы**: ZIP, RAR, 7Z

### Поиск файла
1. Нажмите на **Медиа**
2. Введите имя файла в поле поиска
3. Нажмите Enter

### Отредактировать файл
1. Кликните на файл в библиотеке
2. Отредактируйте альт-текст и описание
3. Нажмите **Сохранить**

## 👥 Пользователи

### Добавить пользователя
1. Admin Panel → **Пользователи** → **Новый пользователь**
2. Заполните форму:
   - Имя: ФИ пользователя
   - Email: `user@example.com`
   - Пароль: безопасный пароль
   - Роль: выберите роль
3. Нажмите **Создать**

### Роли пользователей
- **Администратор** - полный доступ
- **Редактор** - создание контента
- **Автор** - создание собственного контента
- **Подписчик** - только просмотр

### Изменить роль пользователя
1. Admin Panel → **Пользователи**
2. Нажмите на пользователя
3. Измените роль
4. Нажмите **Сохранить**

### Отключить пользователя
1. Admin Panel → **Пользователи**
2. Нажмите **Delete** рядом с пользователем
3. Подтвердите

## 🔍 SEO оптимизация

### Заполнить SEO поля
1. Отредактируйте страницу
2. В правой колонке найдите **SEO**
3. Заполните поля:
   - **Meta Title** (до 60 символов): название для поисковиков
   - **Meta Description** (до 160 символов): описание
   - **Keywords**: ключевые слова через запятую
4. Нажмите **Сохранить**

### Пример SEO полей
```
Meta Title: Услуги веб-дизайна | Your Company
Meta Description: Профессиональный веб-дизайн для малого бизнеса
Keywords: веб-дизайн, дизайн сайта, web-design
```

### Open Graph (для соцсетей)
1. При редактировании страницы прокрутите вниз
2. Заполните:
   - **OG Title** - заголовок для соцсетей
   - **OG Description** - описание
   - **OG Image** - изображение
3. Сохраните

## 💾 Резервные копии

### Создать резервную копию
1. Admin Panel → **Резервные копии** → **Создать резервную копию**
2. Выберите тип:
   - **Полная копия** (БД + файлы)
   - **Только БД**
   - **Только файлы**
3. Нажмите **Создать**

### Восстановить из копии
⚠️ **Внимание**: Это заменит все текущие данные!

1. Admin Panel → **Резервные копии**
2. Найдите нужную копию
3. Нажмите кнопку **Восстановить** (стрелка)
4. Подтвердите

### Скачать копию
1. Admin Panel → **Резервные копии**
2. Найдите копию
3. Нажмите **Download** (стрелка вниз)

## ⚙️ Настройки

### Основные настройки
1. Admin Panel → **Настройки** → **Основные**
2. Заполните:
   - **Название сайта**
   - **Слоган**
   - **Email администратора**
   - **Часовой пояс**
3. Нажмите **Сохранить**

### Безопасность
1. Admin Panel → **Настройки** → **Безопасность**
2. Настройте:
   - Сеансы пользователей
   - Попытки входа
   - Двухфакторная аутентификация
3. Нажмите **Сохранить**

## 🔐 Смена пароля

1. Нажмите на иконку **Профиль** в верхнем углу
2. Выберите **Смена пароля**
3. Введите старый пароль
4. Введите новый пароль (2 раза)
5. Нажмите **Сохранить**

## 📊 Аналитика

### Просмотр статистики
1. Admin Panel → **Главная**
2. Видите карточки с:
   - **Страницы** - количество созданных
   - **Пользователи** - активные пользователи
   - **Файлы** - загруженные файлы
   - **Просмотры** - количество визитов

### Недавняя активность
На главной панели внизу видна **Недавняя активность**:
- Кто создал что
- Когда это произошло
- Какой тип действия

## 🛠️ Решение типовых проблем

### Я забыл пароль
→ Обратитесь к администратору для сброса

### Не могу загрузить файл
→ Проверьте:
- Размер файла (макс. 100 MB)
- Формат файла (поддерживаемый)
- Свободное место на сервере

### Страница не видна на сайте
→ Проверьте:
- Статус: должен быть "Опубликована"
- Видимость: должна быть "Для всех"
- URL: правильно ли введен

### Медленная работа системы
→ Сделайте:
- Очистите кэш (в Settings)
- Проверьте резервные копии
- Свяжитесь с поддержкой

### Ошибка при восстановлении
→ Сделайте:
- Создайте новую резервную копию перед восстановлением
- Попробуйте восстановить из более старой копии
- Свяжитесь с поддержкой

## 🎓 Советы для лучшей работы

1. **Используйте ясные названия страниц**
   - ✅ Хорошо: "Услуги веб-дизайна"
   - ❌ Плохо: "Page 1"

2. **Заполняйте SEO поля**
   - Это улучшит рейтинг в поисковиках
   - Используйте ключевые слова

3. **Создавайте резервные копии**
   - Минимум одна в неделю
   - Проверяйте, что копии создаются

4. **Используйте категории**
   - Организуйте контент по разделам
   - Помогает навигации

5. **Проверяйте орфографию**
   - Перед публикацией прочитайте текст
   - Используйте предпросмотр

6. **Оптимизируйте изображения**
   - Используйте сжатые изображения
   - Правильный размер

7. **Обновляйте регулярно**
   - Свежий контент = больше посетителей
   - Минимум один раз в месяц

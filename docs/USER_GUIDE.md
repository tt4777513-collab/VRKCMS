## Введение

CMS Pro - это мощная и простая в использовании система управления контентом, разработанная специально для малых предприятий. Она позволяет легко создавать, редактировать и публиковать контент без технических знаний.

### Основные возможности

- ✅ Создание и управление страницами
- ✅ Визуальный редактор контента
- ✅ Загрузка и организация медиафайлов
- ✅ Управление пользователями и правами доступа
- ✅ Встроенная оптимизация для SEO
- ✅ Автоматическое резервное копирование
- ✅ Аналитика и статистика

## Управление страницами

### Создание новой страницы

1. Нажмите **"Новая страница"** на вкладке **Страницы**
2. Заполните форму:
   - **Название** - название вашей страницы
   - **URL Slug** - адрес страницы (автоматически из названия)
   - **Содержание** - текст страницы
   - **Категория** - выберите категорию

3. Нажмите **"Сохранить"**

### Редактирование страницы

1. Найдите страницу в списке
2. Нажмите кнопку редактирования (иконка карандаша)
3. Измените нужные поля
4. Нажмите **"Сохранить"**

### Публикация и статусы

Каждая страница имеет три статуса:

- **Черновик** (Draft) - страница создана, но не опубликована
- **Опубликована** (Published) - страница видна всем
- **Архив** (Archive) - архивированная страница

Чтобы опубликовать страницу:
1. Отредактируйте страницу
2. В правой панели установите статус **"Опубликована"**
3. Нажмите **"Сохранить"**

### Видимость страницы

При публикации можно выбрать видимость:
- **Для всех** - доступна всем посетителям
- **Только для зарегистрированных** - видна только авторизованным пользователям
- **Приватная** - видна только вам

### Удаление страницы

1. Найдите страницу в списке
2. Нажмите кнопку удаления (иконка корзины)
3. Подтвердите удаление

1. Выделите текст

1. Нажмите кнопку изображения

4. Нажмите ОК

1. Нажмите кнопку таблицы
2. Укажите количество строк и столбцов
3. Заполните ячейки

## Медиа библиотека

### Загрузка файлов

1. Нажмите **"Загрузить файлы"**
2. Выберите файлы на компьютере (можно несколько)
3. Выберите категорию (опционально)
4. Нажмите **"Загрузить"**

### Поддерживаемые форматы

**Изображения:** JPG, PNG, GIF, WebP, SVG (макс. 10 MB)
**Видео:** MP4, WebM, OGG (макс. 500 MB)
**Документы:** PDF, DOC, DOCX, XLS, XLSX (макс. 50 MB)
**Архивы:** ZIP, RAR, 7Z (макс. 100 MB)

### Организация медиа

1. Используйте фильтры для поиска файлов
2. Сортируйте по дате или размеру
3. Удаляйте ненужные файлы

### Редактирование медиа

1. Нажмите на файл
2. Отредактируйте информацию:
   - Alt-текст (для SEO)
   - Описание
   - Теги
3. Нажмите **"Сохранить"**

## Управление пользователями

### Добавление нового пользователя

1. На вкладке **Пользователи** нажмите **"Новый пользователь"**
2. Заполните данные:
   - **Имя** - ФИ пользователя
   - **Email** - адрес электронной почты
   - **Пароль** - безопасный пароль
   - **Роль** - выберите роль

3. Нажмите **"Создать"**

### Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Администратор** | Полный доступ ко всем функциям |
| **Редактор** | Создание/редактирование страниц и контента |
| **Автор** | Создание собственных страниц |
| **Подписчик** | Только просмотр |

### Редактирование пользователя

1. Нажмите на пользователя в списке
2. Измените данные
3. Нажмите **"Сохранить"**

### Удаление пользователя

1. Найдите пользователя
2. Нажмите кнопку удаления
3. Подтвердите

## SEO оптимизация

### Основные SEO поля

Для каждой страницы заполните:

1. **Meta Title** - название в результатах поиска (до 60 символов)
2. **Meta Description** - описание (до 160 символов)
3. **Meta Keywords** - ключевые слова (разделяйте запятой)

Пример:
```
Meta Title: Услуги веб-дизайна | CMS Pro
Meta Description: Профессиональные услуги веб-дизайна для малого бизнеса
Meta Keywords: веб-дизайн, дизайн сайта, услуги
```

### Open Graph теги

Для социальных сетей заполните:

1. **OG Title** - заголовок для соцсетей
2. **OG Description** - описание для соцсетей
3. **OG Image** - изображение для соцсетей

### URL-адрес (Slug)

- Используйте латинские буквы
- Разделяйте слова дефисом (-)
- Длина: 3-75 символов

Пример: `services-for-small-business`

### Структурированные данные (Schema)

CMS Pro автоматически добавляет Schema.org разметку. Проверьте её в [Schema Validator](https://schema.org/validator/)

## Настройки системы

### Основные настройки

1. Перейдите на вкладку **Настройки**
2. Заполните основные параметры:
   - **Название сайта**
   - **Слоган**
   - **Email администратора**
   - **Часовой пояс**

### Резервные копии

1. На вкладке **Резервные копии** видны все архивы
2. Система автоматически создает ежедневные копии
3. Для ручного создания нажмите **"Создать резервную копию"**
4. Для восстановления нажмите кнопку восстановления

⚠️ **Важно:** Восстановление заменит все текущие данные!

## Аналитика

### Панель управления

На главной странице видна статистика:
- **Страницы** - количество созданных страниц
- **Пользователи** - активные пользователи
- **Файлы** - загруженные медиа
- **Просмотры** - количество визитов

### Недавняя активность

В разделе "Недавняя активность" отображаются последние действия:
- Созданные страницы
- Загруженные файлы
- Добавленные пользователи
- Комментарии

## Частые вопросы

### Я забыл пароль. Что делать?

Обратитесь к администратору для сброса пароля.

### Как увеличить лимит на загрузку файла?

Обратитесь к администратору или хостинг-провайдеру.

### Как поменять пароль?

1. Нажмите на иконку профиля в верхнем углу
2. Выберите "Смена пароля"
3. Введите старый и новый пароль
4. Нажмите "Сохранить"

### Могу ли я восстановить удаленную страницу?

Да, из резервной копии. Обратитесь к администратору.

### Как настроить SEO?

Заполните все поля SEO при редактировании страницы:
- Meta Title
- Meta Description
- Meta Keywords
- Open Graph теги

### Я не вижу мою страницу на сайте. Почему?

Проверьте статус страницы - она должна быть "Опубликована" и видимость "Для всех".

### Какой размер изображения рекомендуется?

- Для главного изображения: 1200x630 px
- Для миниатюр: 400x300 px
- Для галереи: 800x600 px

### Как сделать страницу приватной?

1. Отредактируйте страницу
2. Установите видимость: "Приватная"
3. Сохраните

### Поддерживаются ли видео?

Да, вы можете загружать видео и вставлять YouTube видео по URL.

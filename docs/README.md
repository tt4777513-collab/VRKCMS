# CMS Pro - Информационная система управления контентом

![CMS Pro](https://img.shields.io/badge/version-1.0-blue)
![PHP](https://img.shields.io/badge/PHP-8.0+-green)
![MySQL](https://img.shields.io/badge/MySQL-5.7+-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Профессиональная система управления контентом веб-сайта для малых предприятий. Позволяет сотрудникам без технических знаний создавать, редактировать и публиковать контент через удобный веб-интерфейс.

## 🎯 Основные возможности

- ✅ **Управление страницами** - Создание, редактирование и публикация страниц
- ✅ **Визуальный редактор** - Интуитивный редактор с поддержкой форматирования
- ✅ **Медиа библиотека** - Загрузка и организация изображений, видео, документов
- ✅ **Управление пользователями** - Система ролей и прав доступа
- ✅ **SEO оптимизация** - Встроенная оптимизация для поисковых систем
- ✅ **Резервное копирование** - Автоматическое ежедневное резервное копирование
- ✅ **Аналитика** - Статистика посещений и активности
- ✅ **Администраторская панель** - Полный контроль над сайтом

## 🔧 Полная установка

### Требования
- PHP 8.0+
- MySQL 5.7+
- Composer
- Node.js (для обработки ассетов)

### Процесс установки

1. **Клонирование проекта**
```bash
git clone https://github.com/your-repo/cms-pro.git
cd cms-pro
```

2. **Установка зависимостей**
```bash
composer install
npm install
```

3. **Конфигурация**
```bash
cp .env.example .env
php artisan key:generate
```

4. **Настройка БД**
```bash
# Отредактируйте .env
php artisan migrate
php artisan db:seed
```

5. **Запуск**
```bash
php artisan serve
npm run dev
```

Подробнее см. в [INSTALLATION.md](./INSTALLATION.md)

## 🐛 Решение проблем

### Не видна админ-панель
- Проверьте, что вы находитесь по адресу `/admin`
- Убедитесь, что база данных запущена
- Проверьте файл `.env`

### Ошибка при загрузке файлов
- Проверьте права на папку `/public/uploads`
- Убедитесь в лимите загрузки в PHP
- Проверьте свободное место на диске

### Медленная работа
- Очистите кэш: `php artisan cache:clear`
- Проверьте индексы БД
- Включите Redis кэширование

Более подробную информацию см. в [INSTALLATION.md#-отладка](./INSTALLATION.md)

## 🧪 ТЕСТИРОВАНИЕ

После выполнения скрипта скопируйте эти запросы:

```sql
-- 1. Все клиенты
SELECT CompanyName, Status, ClientCategory FROM Clients;

-- 2. Активные клиенты с контактами
SELECT c.CompanyName, COUNT(*) AS Contacts 
FROM Clients c
LEFT JOIN ClientContacts cc ON c.ClientID = cc.ClientID
GROUP BY c.ClientID, c.CompanyName;

-- 3. Финансовая статистика
SELECT CompanyName, 
       (SELECT COUNT(*) FROM ClientContracts 
        WHERE ClientID = Clients.ClientID) AS Contracts,
       (SELECT SUM(ContractValue) FROM ClientContracts 
        WHERE ClientID = Clients.ClientID) AS TotalValue
FROM Clients;

-- 4. Просроченные платежи
SELECT * FROM vw_OverduePayments;
```

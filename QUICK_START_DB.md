# 🚀 БЫСТРЫЙ СТАРТ - База данных клиентов

## ⚡ 3 шага для запуска

### ✅ Шаг 1: Откройте файл БД
```
database/clients_database.sql
```

### ✅ Шаг 2: Подключитесь к SQL Server
```
Ctrl+Shift+P → "MS SQL: Connect" → выберите сервер
```

### ✅ Шаг 3: Запустите скрипт
```
Ctrl+Shift+E или правый клик → Execute Query
```

## 🔍 Первые запросы

После создания БД скопируйте эти запросы:

```sql
-- Все клиенты
SELECT * FROM Clients;

-- Активные клиенты
SELECT * FROM vw_ActiveClients;

-- Статистика
SELECT * FROM vw_ClientStatistics;

-- Просроченные платежи
SELECT * FROM vw_OverduePayments;
```
